AnimMesh Forge — Help

First Step
Load a sprite & generate a mesh
Click Load Sprite (or drag & drop a PNG). Then choose a mesh type:
• Grid Mesh — uniform N×M grid, best for flags, cloth, water surfaces.
• Alpha Mesh — auto-traces the opaque area, good for characters & irregular shapes.
Adjust cols/rows or Alpha Threshold, then click the button. You can also place vertices manually with Add Vertex mode and hit Triangulate when done.
2
Pose & set keyframes
Switch to Deform mode (D). Drag one or more vertices to pose the mesh. When you release the mouse a keyframe is automatically saved at the current frame.
Move to a different frame on the Timeline, pose again — repeat for each key pose.
You can also press K to manually commit the current pose as a keyframe, or Shift+K to remove the keyframe on the current frame.
3
Edit vertices
• Select mode (V): click to select, Shift+click to multi-select, drag to box-select.
• Add Vertex (A): click anywhere on the sprite to place a new vertex. Mesh re-triangulates automatically.
• Del Vertex: select one or more vertices, then click the button or press Delete/Backspace.
• Ctrl+A selects all. Esc clears the selection.
4
Preview the animation
Press Space to play/stop. Use ◀ ▶ buttons or ← → arrow keys to step through frames. Click anywhere on the Timeline ruler to jump to that frame. The mesh is interpolated between keyframes using the easing curve set in the sidebar.
5
Export as PNG frames in ZIP
Click Export ZIP. Each frame is rendered to a separate PNG (frame_0000.png, frame_0001.png, …) and packed into a ZIP. Scale factor and optional mesh overlay can be set in the Export sidebar section. The export uses pixel-perfect nearest-neighbour sampling — no blending.
